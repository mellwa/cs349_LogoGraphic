Detailed Description

This class encapsulates X lib's Graphics Context (GC) and provides the means by which components draw themselves on windows.

You can add functionality to this class (for example, there are no methods provided to draw arcs, ellipses, or circles).

You may want to add functionality to change fonts and determine the bounds of a string when drawn to the screen

Sets the graphics context's background color. Important: Setting this will have absolutely no visible effect on your window. This color is used when calling XClearArea in Xlib, but we currently do not provide a wrapper method for that function call. Instead, to clear the window to your desired background color, set the foreground color to the desired background color, then use FillRect.
